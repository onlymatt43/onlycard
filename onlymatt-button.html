ONLYMATT Button Effect pour WordPress Breakdance

Copiez ce code HTML dans un élément "Code personnalisé" de Breakdance :

<div class="onlymatt-button"> <div class="photo-overlay" style="background-image: url('VOTRE_URL_IMAGE');"></div> <h1 class="onlymatt-text">ONLYMATT</h1> </div>

Personnalisation :

  • Remplacez VOTRE_URL_IMAGE par l'URL de votre photo
  • Ajoutez subtle pour un effet plus discret
  • Ajoutez minimal pour un look épuré

Exemples :

<!-- Version subtile --> <div class="onlymatt-button subtle"> <div class="photo-overlay" style="background-image: url('image.jpg');"></div> <h1 class="onlymatt-text">ONLYMATT</h1> </div> <!-- Version minimaliste --> <div class="onlymatt-button minimal"> <div class="photo-overlay" style="background-image: url('image.jpg');"></div> <h1 class="onlymatt-text">ONLYMATT</h1> </div>
